# 主流开源AI中转面板清单 · 2026-04

所有这些都能 Docker 一键部署，选一个适合自己规模的。

## new-api
- GitHub: github.com/Calcium-Ion/new-api
- 语言: Go + React
- 数据库: MySQL / PostgreSQL
- 特点: 功能最全，支持50+渠道，中文社区最活跃
- 部署: 资料包 `install.sh` 里选 [2]
- 适合: 准备做成规模工作室的人

## one-api
- GitHub: github.com/songquanpeng/one-api
- 语言: Go
- 数据库: MySQL / SQLite
- 特点: new-api的前身，极简稳定，起步首选
- 部署: 资料包 `install.sh` 里选 [3]
- 适合: 个人小规模试水

## Sub2Api
- 作者: weishaw
- 特点: 订阅制中转，分组+倍率定价灵活
- 部署: 资料包 `install.sh` 里选 [1]
- 适合: 想做差异化定价的玩家

## 其他值得关注
- **openai-forward** (KenyonY/openai-forward): 单文件转发，无UI，适合纯API路由场景
- **NextChat** (ChatGPTNextWeb/NextChat): 只做前端UI，跟中转面板配合
- **chatgpt-web-midjourney-proxy**: 带MJ/SD集成的增强版

## 跑之前请确认

1. VPS 已装好 Docker（`install.sh` 会自动装）
2. Cloudflare WARP 已接管出境流量（`install.sh` 会自动装）
3. 域名已托管给 Cloudflare + Tunnel token 已拿到
4. 数据库（Neon / MySQL）+ 缓存（Upstash / Redis）已就绪

其中 1、2 脚本自动处理，3、4 要你手动去各家面板准备。
