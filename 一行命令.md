# 一行命令部署

在 Ubuntu 22.04/24.04 LTS 的 root 用户下执行下面这行命令，自动装 Docker + WARP + 交互式选择面板：

```bash
bash <(curl -fsSL https://你的资源托管/install.sh)
```

> 上面这行命令需要你先把资料包里的 `install.sh` 上传到你自己的 OSS / GitHub Gist / 公众号服务器
> 然后把 URL 替换进去。这样读者就能一行命令起环境。

## 零托管方案

如果你不想托管脚本，直接让读者把 `install.sh` 下载到服务器上跑：

```bash
wget https://你的公众号/install.sh && chmod +x install.sh && ./install.sh
```

或者用 scp 从本地上传：

```bash
scp install.sh root@你的VPS_IP:~ && ssh root@你的VPS_IP 'bash install.sh'
```

## 进阶：非交互式部署（脚本串联）

```bash
PANEL=new-api bash install.sh  # 预设面板名跳过交互
```
